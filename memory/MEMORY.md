## テキストクラスルール（assets/CLAUDE.md 更新済み）

`.text-sub` は廃止。以下の3つで代替する：

| クラス／要素 | 用途 |
|---|---|
| `<p>` | 本文・説明文・指示文。迷ったらこれ |
| `.text-note` | 補足・注釈（読まなくても話が通じる情報）|
| `.next-box` | 次回予告・引き継ぎ・保存指示 |

## SVGサイズの教訓（024で確認）

小さく見える SVG の修正パターン：
- viewBox を縮小（例：1300→900）→ スケール係数が上がり文字が大きく見える
- max-height を増加（例：420px→680〜720px）→ 縦方向のスペースを活用
- フォントサイズも合わせて引き上げる（13px→16px、15px→17px 等）

## クイズデータのキー名（重要）

`slide-template.js` は `q.choices` を参照する。生成するクイズデータは必ず `choices:` を使うこと。
`options:` は旧形式（全ファイル `choices` に統一済み）。
