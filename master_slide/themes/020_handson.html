ワーク①：5カテゴリ自己評価

目標

第1か月で学んだ19テーマを5カテゴリに分け、各カテゴリの習熟度を1〜5で自己採点します。

評価基準

  • 1：ほとんど理解できていない / 実践できない
  • 2：概念は理解したが、自信を持って使えない
  • 3：基本的なことはできる / 説明もある程度できる
  • 4：自信を持って使える / 他者に教えられる
  • 5：応用も自在 / 自分なりの工夫ができる
⏱️ 制限時間：5分 / 💡 直感で採点してOK

5カテゴリの評価項目

各カテゴリで問いかける質問

  • 設計思考（1〜5）：「このデザインは なぜ 良いのか？」を論理的に説明できるか
  • 表現技術（1〜5）：Figma / Canvaで意図通りの形を作れるか
  • AI活用（1〜5）：AIに適切な指示を出して、狙った成果物を得られるか
  • 実践力（1〜5）：制限時間内に80点のクオリティを出せるか
  • 統合力（1〜5）：複数の原則・要素を組み合わせて判断できるか
📝 紙またはメモアプリに「設計思考：3」のように記録してください

ワーク②：4象限マッピング

目標

19テーマを「好き/苦手意識」×「得意/苦手」の4象限に配置し、戦略的な優先順位をつけます。

4象限の意味

  • 左上（好き×得意）：強み → 100日実践の「武器」として活用
  • 右上（苦手意識×得意）：可能性 → 思い込みを外す
  • 左下（好き×苦手）：成長余地 → 100日で重点強化
  • 右下（苦手意識×苦手）：外注候補 → AIや他者に任せる
⏱️ 制限時間：10分 / 📝 紙に4象限を描いてテーマを配置

配置する19テーマ

各テーマを4象限のどこに置くか考える

  • デザインは「設計」である / 生成AI完全制御 / 色彩と文字
  • ブランド・アイデンティティ / Figmaによる骨格設計
  • Atomic Designとコンポーネント / CRO理論
  • モバイルファースト / 写真加工（Canva×AI）
  • SNS特化型デザイン / LPのビジュアル戦略
  • ポートフォリオと人間の感性 / 言語化ドリル（1）（2）
  • カスタムAI（Gem）構築 / 80点を30分でタイムアタック
  • ロジック・プロンプト / クライアント提案シミュレーション
  • 最新AIツール実験室
💡 全部配置しなくてOK。特に印象に残っているテーマから始めましょう

週単位の計画を立てる

ワーク④：100日マイルストーン

目標

100日を「4つのフェーズ」に分け、各フェーズの達成目標を設定します。

100日の4フェーズ

  • Day 1-25（準備期）：環境構築 / 最初の一歩を踏み出す
  • Day 26-50（実行期）：本格的に取り組む / 最初の成果を出す
  • Day 51-75（加速期）：フィードバックを反映 / 量をこなす
  • Day 76-100（仕上げ期）：成果を形にする / 振り返りと次の計画
⏱️ 制限時間：10分 / 📝 各フェーズに1つの達成目標を設定

フェーズ別目標の例

クラウドソーシングを選んだ場合

  • 準備期：プロフィール作成 / ポートフォリオ3点用意 / 最初の提案を送る
  • 実行期：初案件を受注・納品 / 評価を獲得
  • 加速期：3件目の案件を完了 / リピーター獲得を狙う
  • 仕上げ期：5件以上の実績 / 月1万円以上の収入

イベント開催を選んだ場合

  • 準備期：コンテンツ企画 / 集客チャネル決定 / 告知開始
  • 実行期：初回イベント開催 / 参加者フィードバック収集
  • 加速期：2回目開催 / 改善点を反映
  • 仕上げ期：3回目開催 / コミュニティ形成の基盤完成
