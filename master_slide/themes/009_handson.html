演習1
人物写真 → 白背景ポートレート

自撮りをプロフィール写真レベルに変換

Step 1：Geminiで加工方針を検討

なぜ先にGeminiを使うのか？

いきなり加工を始めると、「何をどこまで加工すべきか」が曖昧になります。
Geminiで方針を言語化してから作業することで、判断がブレません。

Geminiに入力するプロンプト

以下の人物写真をプロフィール用に加工したいです。 【現状】 - 撮影場所：自宅のリビング（生活感のある背景） - 照明：自然光（窓から） - 服装：ビジネスカジュアル 【目標】 - 用途：LinkedInプロフィール / 社内ポータル - 印象：清潔感、信頼感、親しみやすさ 以下を提案してください： 1. 背景をどう処理すべきか（白/グレー/ぼかし） 2. 残すべき要素と削除すべき要素 3. 加工の優先順位（まず何から着手すべきか） 4. 注意すべきポイント（不自然になりやすい箇所）
⚠️ 「きれいにして」では曖昧すぎる。用途と目標を具体的に伝える

Step 2：Geminiの回答を判断基準に変換

Geminiの回答例と解釈

背景処理の提案：

  • 「白背景」→ Canvaの背景リムーバー + 白塗りつぶし
  • 「ぼかし」→ 背景リムーバー + グラデーション背景

削除すべき要素：

  • 生活感のあるもの（家具、雑貨、洗濯物）
  • 他の人物が写り込んでいる場合

残すべき要素：

  • 人物の輪郭・髪の毛（自然なエッジを保つ）
  • 服装のディテール
💡 Geminiの回答を「Canvaでの操作手順」に翻訳する

ツール紹介：Canva

初回アクセス：

https://www.canva.com/

今日使う機能：

  • 背景リムーバー：人物を自動で切り抜き
  • 生成塗りつぶし：背景の差し替え・拡張
  • マジック消しゴム：不要オブジェクトの削除
💡 Pro版が必要ですが、無料トライアルで試せます

手順 1 / 3：背景リムーバーで切り抜き

操作手順

1. Canvaで新規デザイン作成（1080×1080px または用途に合わせたサイズ）

2. 自撮り写真をアップロード

3. 写真を選択 →「写真を編集」→「背景リムーバー」をクリック

4. 自動処理完了を待つ（数秒）

理論編との関連

背景リムーバーは「背景全体の差し替え」を可能にします。
理論編で学んだ「Level 3：大規模な変更」に該当しますが、人物切り抜きはAIが得意な処理です。

⚠️ よくある間違い：髪の毛のエッジが雑なまま進める
✅ 正解：「復元」ブラシで髪の毛の境界を丁寧に調整

手順 2 / 3：白背景を追加

操作手順

1. 左メニュー「素材」→「図形」→ 四角形を選択

2. キャンバス全体を覆うサイズに拡大

3. 色を「白（#FFFFFF）」に変更

4. 右クリック →「背面へ移動」→「最背面へ」

プロフェッショナルに見せるコツ

  • 純白 vs オフホワイト：純白（#FFFFFF）は清潔感、オフホワイト（#F5F5F5）は柔らかさ
  • グラデーション：上部を白、下部を薄いグレーにすると奥行きが出る
⚠️ よくある間違い：背景と人物の境界がくっきりしすぎる
✅ 正解：影を追加するか、わずかにぼかしを入れて自然に

手順 3 / 3：影を追加して自然に仕上げる

操作手順

1. 切り抜いた人物画像を選択

2. 「写真を編集」→「影付き」エフェクトを選択

3. 影の強度を「20〜30%」程度に調整（自然さを優先）

4. 全体を確認し、書き出し（PNG推奨）

スタジオ撮影との違いを埋める

  • 照明の一貫性：元写真の光の方向と影の方向を合わせる
  • コントラスト：必要に応じて明るさ・コントラストを微調整
💡 「スタジオ撮影っぽさ」は影と光の一貫性で決まる

演習2
商品写真 → 高級感のある黒背景

スマホ撮影をEC商品ページレベルに変換

Step 1：Geminiで加工方針を検討

商品写真特有の考慮点

人物写真と違い、商品写真は「商品そのものの質感」を正確に伝える必要があります。
過度な加工は「実物と違う」クレームの原因になります。

Geminiに入力するプロンプト

以下の商品写真を高級感のあるEC商品ページ用に加工したいです。 【現状】 - 商品：[腕時計 / アクセサリー / 化粧品など] - 撮影環境：自宅のテーブル上（木目背景） - 照明：室内照明（やや黄色がかっている） 【目標】 - 用途：ECサイトの商品ページ / SNS広告 - 印象：高級感、洗練された雰囲気、購買意欲を刺激 以下を提案してください： 1. 背景の処理方法（黒背景 / グラデーション / 反射面） 2. 商品の質感を損なわない加工の限界 3. 高級感を演出するための追加要素（影、光沢など） 4. 避けるべき加工（商品の信頼性を損なうもの）
⚠️ 商品カテゴリを具体的に伝えることで、適切な提案が得られる

Step 2：商品写真加工の判断基準

許容される加工

  • 背景の差し替え：商品の色味を変えない範囲で
  • 影の追加：商品の形状を際立たせる
  • 不要物の削除：ホコリ、指紋、背景の雑貨
  • 明るさ・コントラスト調整：実物に近づける範囲で

避けるべき加工

  • 商品の色を変える：実物と違うとクレームに
  • 傷や汚れを消す：中古品の場合は詐欺になる
  • サイズ感を変える：誤解を招く
🎯 理論編の倫理的考慮：「この加工は購入者を欺くか？」

手順 1 / 3：商品を切り抜く

操作手順

1. Canvaで新規デザイン作成（1200×1200px：ECサイト推奨サイズ）

2. 商品写真をアップロード

3. 「写真を編集」→「背景リムーバー」

4. 商品のエッジを確認し、「消去」「復元」ブラシで調整

商品写真特有の注意点

  • 透明な部分：ガラス、宝石などは自動切り抜きが苦手
  • 反射：金属製品の反射は手動で調整が必要
  • 細いパーツ：アクセサリーのチェーンなどは慎重に
⚠️ よくある間違い：自動切り抜きの結果をそのまま使う
✅ 正解：必ずエッジを拡大して確認・調整

手順 2 / 3：高級感のある黒背景を作成

操作手順

1. 左メニュー「素材」→「図形」→ 四角形を選択

2. キャンバス全体を覆うサイズに拡大

3. 色を設定：

  • 純黒：#000000（モダン、シャープ）
  • ダークグレー：#1a1a1a（やや柔らかい）
  • グラデーション：上部 #2a2a2a → 下部 #000000（奥行き感）

4. 右クリック →「最背面へ」

高級感を高めるテクニック

商品の下に微かな反射面を作ると、宙に浮いて見えず自然になります。

💡 黒背景は商品の色が映えるが、暗い商品は沈んでしまう

手順 3 / 3：影と反射で高級感を演出

操作手順（影の追加）

1. 切り抜いた商品画像を選択

2. 「写真を編集」→「影付き」

3. 影の設定：

  • 方向：下方向（商品が浮いているように）
  • ぼかし：中程度（シャープすぎると不自然）
  • 距離：小さめ（商品に近い影）

操作手順（反射面の作成 - 上級）

1. 商品画像を複製

2. 複製を上下反転 →「透明度」を20%程度に

3. 元画像の下に配置（反射に見える）

⚠️ よくある間違い：影を強くしすぎてコントラストが崩れる
✅ 正解：影は「存在を感じる」程度の控えめさ
