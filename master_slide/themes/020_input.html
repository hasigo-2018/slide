「伸びしろ」は「機会」である

苦手を「弱点」と捉えない

第1か月で苦手に感じた分野があっても、それは「まだ練習が足りない」だけ。100日実践は、その「伸びしろ」を実戦で埋める期間です。

戦略的な「伸びしろ」の扱い方

  • 必須スキル：100日で重点的に強化する（判断力に直結）
  • 補助スキル：AIや外注で補完する（効率化の対象）
  • 不要スキル：今は捨てる（選択と集中）
⚠️ すべてを完璧にしようとしない。「勝てる領域」に集中する。

Phase 3
戦略の設計

100日で何を「カタチ」にするか

100日実践とは何か

「学び」を「実績」に変換する期間

第1か月で得た知識と言語化力を、実際のプロジェクトで使い、「判断力」を磨く期間です。

100日で達成すべきこと

  • アウトプット：自分がやりたいことを「カタチ」にする
  • フィードバック：他者からの評価を受け、改善する
  • 判断力の獲得：「良い/悪い」を自分で決められるようになる
🎯 ゴール：「社内でデザイン判断ができる人材」になる

100日実践の「4つの道」

4つの道の特徴

自分に合った道を選ぶための判断基準

  • クラウドソーシング：短期で成果を出したい / フィードバックを多く得たい / 実績を積みたい
  • 起業準備：自分のビジョンを形にしたい / 長期的な資産を作りたい
  • イベント開催：人と繋がりたい / 教える側に回りたい / コミュニティを作りたい
  • クラウドファンディング：アイデアを市場で検証したい / 資金と注目を同時に得たい
💡 複数の道を組み合わせることも可能。ただし「主軸」を1つ決める。

クラウドソーシングを選ぶなら

4つのプラットフォームの特徴

  • クラウドワークス：案件数最多 / 競争激しい / 実績構築向き
  • ランサーズ：認定制度あり / 継続案件を狙いやすい
  • ココナラ：自分から出品 / 価格設定の自由度高い / ブランディング向き
  • Upwork：海外市場 / 単価高い / 英語必須
⚠️ 「勝てる市場」を選ぶ：自分の強み × 市場のニーズ のマッチングが重要

100日戦略を設計する「3つの問い」

自分に問いかけるべきこと

  • 問い1：何をカタチにしたいか？
    — クラウドソーシングでの実績？ 自分のサービス？ コミュニティ？
  • 問い2：どの強みを武器にするか？
    — 設計思考？ AI活用？ スピード？ 言語化力？
  • 問い3：100日後、どんな自分になっていたいか？
    — 「デザイン判断ができる人」として、どんな場面で活躍しているか？
📝 この3つの問いへの答えが、100日実践の「設計図」になる
