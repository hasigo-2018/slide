# 構成指示書：001 デザインは「設計」である【実践編】

## 基本情報
- **テーマ:** デザインは「設計」である
- **目的:** 実在するLPを「設計の視点」で分析し、「なぜ結果が出るか」を自分の言葉で言語化する
- **推奨スライド数:** 10〜12枚
- **推奨時間:** 約25分 + 演習20分
- **使用ツール:** Gemini（画像分析・構造分析）
- **実例:** キャリアリンクファクトリーLP 2本（理論編と同じ）

## 重要な設計意図
- 旧実践編のApple/Nike/Coca-Colaは **削除**。グローバルブランドではなく、コースの出口（LP/HP制作）に直結する実在LP素材を使用
- 「自分の言葉で100文字で書く」ステップを必ず含める（Block 5の100文字エビデンスへの伏線）
- Geminiの分析を「見ずに」自分で書く → その後比較、というパターンを確立する（002〜005でも継続）

## 参考サイト（設計品質の基準値として参照）

講師が演習中に「良いLPとはどういうものか」の基準値を示す際、以下を参照先として活用できる。

- ベイジ制作：にんべん・サイボウズ採用サイト（BtoB寄りの情報設計が参考になる）
- シフトブレイン制作：LayerX・美郷町（ブランドとビジュアルの一貫性が参考になる）
- クオートワークス制作実績（コーポレート系の導線設計が参考になる）
- SaaS系LP：SmartHR・Sansan・Backlog（Block 4以降のTALENTBOARDと同領域）
