# 構成指示書：005 行動心理学とコンバージョンデザイン【実践編】

## 基本情報
- **テーマ:** 行動心理学とコンバージョンデザイン
- **目的:** 実在するLP・ECサイトから3つのバイアスが「どこに、どう使われているか」を発見し、言語化する
- **推奨スライド数:** 12〜14枚
- **推奨時間:** 約25分 + 演習20分
- **使用ツール:** Gemini
- **パターン:** 分析・言語化型
- **実例:** キャリアリンクファクトリー2回目LP ＋ 生徒各自が選んだECサイト

## 重要な設計意図
- Block 1の集大成。001で初めて見たキャリアリンクファクトリーLPを「行動心理学の目」で再分析
- 001では「なぜ設計が違うと結果が違うのか」、005では「その設計の中にどんな心理学が使われているか」
- 同じ素材を視点を変えて見ることで、分析の解像度が回を追うごとに上がる体験
- 「自分の言葉で100文字」パターンの5回目（習慣化の完成）
- Block 1全体の振り返りスライドを含む

## 参考サイト（設計品質の基準値として参照）

Block 1の集大成として、行動心理学が効果的に組み込まれた実在サイトを参照先として活用できる。

- ベイジ制作：にんべん（BtoC寄り。社会的証明の見せ方が参考になる）
- SaaS系LP：SmartHR・Sansan（BtoB SaaSでの損失回避・社会的証明の使い方。TALENTBOARDと同領域で、Block 5以降の伏線にもなる）
- クオートワークス制作実績（CTAまでの導線に行動心理学が組み込まれている事例が多い）

※005の演習素材はキャリアリンクファクトリーLP＋受講生が選ぶECサイト。上記はBlock 1全体の振り返り（スライド11）で「今後参考にすべきプロの制作物」として紹介する位置づけ。

## クイズ（5問・実践判断型）

| No. | 問題 | 正解のポイント |
|-----|------|---------------|
| Q1 | キャリアリンクファクトリーLPの「管理者付きチーム派遣」「コンプライアンス徹底」は、どのバイアスを活用しているか？ | 損失回避（導入によるリスク＝損失を払拭している） |
| Q2 | ECサイトの「残り3点」表示が効果的な理由を、2つのバイアスで説明すると？ | 損失回避（手に入れないと失う感覚）＋社会的証明（他の人が買っているから残りわずか） |
| Q3 | 「通常価格¥12,800 → 特別価格¥9,800」の表示で最初に見せる¥12,800の役割は？ | アンカー（比較の基準点を設定し、¥9,800をお得に感じさせる） |
| Q4 | 3つのバイアスを探す際、「見つからない」場合に考えるべきことは？ | そのサイトがバイアスを使っていない可能性と、自分の分析観点が足りない可能性の両方を検討する |
| Q5 | Block 1で学んだ5つの武器のうち、LP制作の「情報の順番」を決める際に最も直接的に使うのは？ | 視線誘導（F/Z法則）と認知負荷の管理（003で学んだ内容） |
