# 構成指示書：021 AI時代の著作権・商用利用・プロの防衛術【実践編】

## 基本情報
- **テーマ:** AI時代の著作権・商用利用・プロの防衛術
- **目的:** 016〜020で生成したTALENTBOARDの全素材を著作権チェックリストで検証し、クライアントへの説明文を作成する
- **推奨スライド数:** 8〜10枚
- **推奨時間:** 約50分×1コマ（50分）
- **使用ツール:** Gemini（類似度チェック・説明文作成）
- **ケーススタディ:** TALENTBOARD（016〜020で生成した全素材）

## 重要な設計意図
- Block 4で作った「自分の成果物」を自分で検証する。他人事ではなく自分事として著作権を体験させる
- 「チェックリストを埋める」作業を通じて、理論編の知識を「判断の習慣」に変換する
- クライアント説明文の作成は、Block 6「論理的言語化」への伏線。法的リスクを非専門家に平易に説明する訓練
- 1コマなので演習はコンパクト。チェックリスト検証＋説明文作成の2本立て
