# 構成指示書：015 Figma AI機能の活用【実践編】

## 基本情報
- **テーマ:** Figma AI機能の活用
- **目的:** SŌILのTOPページWFに対してFigma AI＋プラグインで「もう1案作る」「改善する」「プラグインを試す」の3演習を行い、AI出力の3軸評価力を実践する
- **推奨スライド数:** 8〜10枚
- **推奨時間:** 約15分 + 演習45分
- **使用ツール:** Figma（公式AI機能＋サードパーティプラグイン）
- **ケーススタディ:** SŌIL（Block 3共通）

## 重要な設計意図
- 演習の目的は「速く作ること」ではなく「AI出力を評価する力をつけること」
- 演習①のAI生成物の3分類（使える/参考になる/使えない）はBlock 6の035（選球眼）の予行演習
- 演習②はWF全体をAIで作り直すのではなく、1セクション単位で改善提案を受ける＝実務的な活用パターン
- 具体的なプラグイン名はスライドに固定しない。講師が授業前に選定してURL共有する運用
- Block 3の「まとめ」テーマでもある。011〜014の成果物を振り返り、達成感を演出する

## 理論編からの踏襲ポイント
- AI出力の3軸評価基準：①設計意図との整合性 ②コンポーネント設計との一貫性 ③実装可能性
- AIが得意な工程（生成・配置提案・パターン出し）と人間がやるべき工程（優先順位判断・トンマナ整合性・設計意図の言語化）
- 「AIが生成したものをそのまま納品しない。3軸でチェックし、設計意図を言語化してから使う」
- プラグインの選定基準：★評価数・ダウンロード数・最終更新日

## ビジュアルアーキテクト育成上の位置づけ

このテーマは、題材「Figma AI機能の活用」を通じて、ビジュアルアーキテクトに必要な「目的に沿って設計・判断・説明する力」を鍛える位置づけである。単に作業手順やツール操作を覚えるのではなく、このテーマの目的（SŌILのTOPページWFに対してFigma AI＋プラグインで「もう1案作る」「改善する」「プラグインを試す」の3演習を行い、AI出力の3軸評価力を実践する）に向けて、何を選び、なぜそうするかを自分の言葉で説明できる状態をゴールとする。

## 視覚化方針

このテーマでは、文章説明だけでなく、必要に応じてSVG図解・画像生成・既存アセットを使い分ける。特に、初学者が状況を想像しづらい箇所、Before/After、判断基準、成果物イメージ、実務での使用場面は視覚化候補として扱う。スライド生成時は「画像があると理解が速くなるか」「SVGで構造化した方がよいか」「既存素材で具体化できるか」を確認し、視覚要素が学習目的を支える場合に採用する。
