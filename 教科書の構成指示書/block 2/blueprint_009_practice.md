# 構成指示書：009 サイトマップと導線設計【実践編】

## 基本情報
- **テーマ:** サイトマップと導線設計
- **目的:** 整骨院サイトのサイトマップ（LP型・複数ページ型の2案）と導線を設計する
- **推奨スライド数:** 10〜12枚
- **推奨時間:** 約20分 + 演習30分
- **使用ツール:** Gemini, Figma
- **実例:** 007から継続の整骨院案件

## 重要な設計意図
- LP型と複数ページ型の「両方作って比較→1つ選ぶ」が演習の核。判断力の訓練
- 導線の矢印をFigmaで実際に描くことで、抽象的な「導線設計」を可視化する
- 離脱ポイント対策は008のジャーニーマップの「感情」行を直接参照する
- 010への接続：009のサイトマップは010でトンマナを適用する土台になる
