# 構成指示書：010 ブランド・アイデンティティとトンマナの言語化【実践編】

## 基本情報
- **テーマ:** ブランド・アイデンティティとトンマナの言語化
- **目的:** 整骨院サイトの簡易スタイルガイドを作成し、Block 2の上流設計6点セットを完成させる
- **推奨スライド数:** 10〜12枚
- **推奨時間:** 約20分 + 演習30分
- **使用ツール:** Gemini, Figma
- **実例:** 007から継続の整骨院案件（Block 2最終演習）

## 重要な設計意図
- Block 2の集大成。007〜010の全成果物を使う演習
- Geminiプロンプトは2回：①カラー＋フォント ②写真＋言葉遣い。1回でまとめると回答が浅くなる
- 余白・レイアウトはAIに頼らず自分で決めさせる（アーキテクトの判断力訓練）
- 最後に6点セットを並べて「上流設計の完成」を体験させる
- 整骨院案件はここで完結。Block 3以降で別案件に切り替える

## クイズ（5問・実践判断型）

| No. | 問題 | 正解のポイント |
|-----|------|---------------|
| Q1 | カラーパレットを「青にしよう」と決める前に確認すべきことは？ | ペルソナ・JTBDから逆算して、その色がターゲットに適切な印象を与えるか |
| Q2 | 余白ルールをAIに任せず自分で決めるべき理由は？ | 余白は009のサイト構造・情報密度と直結するため、自分の設計判断が必要 |
| Q3 | Geminiが提案したフォントが「明朝体」だった場合、どう判断する？ | ペルソナの年齢層・デバイス環境で読みやすいか、007の好み・NGと矛盾しないかで判断 |
| Q4 | 簡易スタイルガイドに含めるべき最低限の要素は？ | カラーパレット（HEXコード付き）・フォント見本・写真のOK/NG基準・言葉遣いルール |
| Q5 | Block 2の上流設計が完成したことで、Block 3以降の制作がどう変わる？ | 設計の判断基準が明確なため、「なんとなく」の迷いがなくなり、制作スピードと品質が上がる |
