# 構成指示書：033 【公開添削①】他者のデザインを心理学・原則で論理的に斬る【理論編】

## 基本情報
- **テーマ:** 【公開添削①】他者のデザインを心理学・原則で論理的に斬る
- **目的:** 初見の外部LPに対して、Block 1–5の知識を総動員した論理的な分析・添削ができる力を身につける
- **推奨スライド数:** 10〜12枚
- **推奨時間:** 約50分×1コマ（50分）
- **使用ツール:** Gemini（構造分析補助）、Figma（赤入れ演習で使用）
- **素材:** 講師が選定した外部の実在LP（改善余地のあるもの）2本

## 重要な制約
- 030のエビデンス3層構造・4軸評価は **既知前提**
- 031の翻訳5類型・3ステップは **既知前提**
- 032の赤入れ4要素・優先度タグは **既知前提**
- A/Bテスト・データに基づく判断は **含めない**（034で扱う）
- 受講生同士の添削は **含めない**（036で扱う）
- このテーマは「初見の他者のデザインを論理的に斬る」ことに100%集中する
- 030–032はすべて「自分の作品」が対象だった。033で初めて「他者の作品」に刃を向ける。心理的ハードルと倫理的作法の両方を扱う

## 外部LP選定基準（講師用）

| 基準 | 内容 |
|------|------|
| 業種 | BtoB SaaS推奨（TALENTBOARDと同業種だと比較分析しやすい）。BtoCでも可だが、受講生のBlock 4–5の経験値と離れすぎないこと |
| 改善余地 | 「致命的に悪い」ではなく「部分的に改善できる」レベル。全否定の演習にならないよう、良い点と改善点が両方あるLPを選ぶ |
| 本数 | 2本。1本目は分析のデモ＋全員で練習。2本目は個人演習。同業種の競合同士だと比較軸が明確になる |
| 確認事項 | 公開されているLPであること（スクリーンショットの教育利用の範囲内）。特定の企業を誹謗中傷する文脈にならないよう配慮 |
| 選定のコツ | 001実践編の伏線「悪いLPを見つけて診断する」を回収する位置。FV・導線・CTA・色彩のいずれかに明確な改善余地があるLPが望ましい |

## クイズ（5問）

| No. | 問題 | 正解のポイント |
|-----|------|---------------|
| Q1 | 公開添削の3原則は？ | ①良い点を先に挙げる ②指摘には改善案を添える ③人格と作品を分離する |
| Q2 | 添削の5ステップは？ | 俯瞰→設計分析→セクション別評価→改善提案→赤入れ |
| Q3 | 他者のLPを分析するとき、自分のLP（TALENTBOARD）の分析と最も大きく異なる点は？ | 設計意図が不明であること。自分の作品は「こう意図した」と言えるが、他者の作品は「おそらくこう意図したのだろう」と推測から始める必要がある。だからこそ知識ベース（引き出しマップ）での分析が重要 |
| Q4 | 添削で「良い点を先に挙げる」理由は？ | ①全否定は分析力の欠如を示す（どんなLPにも設計意図はある）。②036で受講生同士の添削をするとき、受け手の心理的安全を確保するための作法。③実務でクライアントのデザインを添削する際にも必須のコミュニケーション技術 |
| Q5 | 033の添削レポートが034以降にどうつながるか？ | 033で「論理的に斬る力」を身につけた後、034でA/Bテストによるデータ検証、035で大量の案から1案を選ぶ「選球眼」、036で受講生同士の添削と段階的に応用が広がる。033は「判断→言語化→提案」の総合演習であり、034–036の全テーマの前提になる |
